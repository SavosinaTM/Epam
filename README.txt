Разработать систему управления, используя многопоточность.
Контекст задачи.
Порт. Корабли (ships) заходят в порт для разгрузки (load)/загрузки (unload).
Работает несколько причалов (docks). У одного причала может стоять один корабль.
Корабль может загружаться у причала, разгружаться или выполнять оба действия.
